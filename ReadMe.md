## To-do List

Uma lista de tarefas onde podemos utilizar um campo para inserir, marcar e 
excluir as pendências do cotidiano.

#### Tecnologias

- HTML5
- CSS3
- JavaScript

#### Demonstração

https://ja-lourenco.github.io/to-doList/

#### Funcionalidades

- Multiplataforma

#### Aprendizados

- Manipulação de DOM
- Eventos
- Local Storage
- JSON básico
- Arrays: push, splice e indexOf
